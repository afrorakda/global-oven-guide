# #62 Global Oven Guide

A hyper-minimalist oven temperature converter designed for global home cooks. It instantly synchronizes Celsius, Fahrenheit, and Gas Mark, while providing essential adjustments for Fan (Convection) ovens.

## Core Philosophy
* **1 Tool / 1 Action**: Enter one value, and all other scales update instantly.
* **Maman-First Design**: Focused on solving the common "oven scale confusion" found in international recipes.
* **Practical Accuracy**: Instead of complex decimals, it provides rounded values that match actual oven dials.

## Key Features
* **Triple-Scale Sync**: Simultaneous conversion between Celsius (°C), Fahrenheit (°F), and Gas Mark (1-9).
* **Fan Oven Logic**: Automatically calculates the recommended temperature for Fan/Convection ovens (approx. -20°C / -25°F).
* **Smart Reset**: Any "clr" button clears the entire board to prevent calculation mix-ups, keeping the workspace clean.
* **Mobile Optimized**: Featuring a large numeric keypad trigger and 15px rounded-corner UI for easy operation in the kitchen.

## Design System
* **Background**: #fff
* **Main Accent**: #333
* **Border Radius**: 15px
* **iOS Optimization**: Enhanced visibility for inputs and prevented auto-zoom issues.

## Privacy & Performance
* **No Tracking / No Ads / No Login**
* **Browser-only**: Works offline once loaded.
* **High Performance**: Zero dependencies, lightweight HTML/JS.
